# 📘 Student AI Pro – Intelligent Learning Chatbot

A modern AI-powered educational chatbot built with **Streamlit**, designed to help students learn, interact, and manage conversations efficiently.

---

## 🚀 Features

- 🧠 **AI Chat Assistant** (Groq – LLaMA 3)
- 💬 **Multi-Chat System**
- 🔐 **User Authentication (Login/Register)**
- 🧾 **Persistent Chat History (SQLite)**
- 🧠 **Smart Memory System** (remembers user info like name)
- 📎 **File Upload Support**
  - PDF text extraction
  - TXT file reading
  - Image OCR (Tesseract)
- 🎤 **Voice Input (Speech Recognition)**
- 🌐 **Web Search Integration (DuckDuckGo)**
- 🎭 **Multiple AI Personas**
  - 📘 Tutor
  - 🤖 Assistant
  - 🎨 Creative Thinker
  - 🎯 Exam Coach
- 🎨 **Modern UI (Custom Styled Streamlit)**
