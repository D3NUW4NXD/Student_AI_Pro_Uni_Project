# 🎓 Student AI Pro - University Project

An AI-powered student assistant system designed to enhance academic productivity and provide intelligent support for students. This project integrates modern Artificial Intelligence techniques to deliver smart features such as chatbot assistance, automation, and data-driven insights.

---

## 🚀 Project Overview

Student AI Pro is a university-level AI application developed to assist students in their daily academic tasks. The system leverages AI technologies to improve efficiency, reduce manual effort, and provide a smarter learning experience.

---

## ✨ Key Features

- 🤖 AI Chatbot for student interaction
- 🧠 Natural Language Processing (NLP) support
- 📊 Data analysis and insights
- 🔍 Smart query handling
- 💬 Real-time responses
- 🌐 User-friendly interface
- 🔐 Secure system design

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/D3NUW4NXD/Student_AI_Pro_Uni_Project.git
cd Student_AI_Pro_Uni_Project
```

### 2. Create virtual environment
```bash
python -m venv venv
```

### 3. Activate environment

**Windows**
```bash
venv\Scripts\activate
```

**Mac/Linux**
```bash
source venv/bin/activate
```

### 5. Run the app
```bash
python app.py
```
